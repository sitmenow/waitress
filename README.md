# Waitress
Sit Me Now application service.


## Setup

### Requirements
Install nodejs + yarn:
```bash
$ brew install nodeenv
$ brew install yarn --without-node

$ nodeenv /path/to/your/envs/node
$ source /path/to/your/envs/node/bin/activate
$ cd /path/to/waitress/repo/path
$ yarn install
```

## Test
